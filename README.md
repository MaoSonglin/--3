##### 跟踪甘特图
> 跟踪甘特图是在项目计划上增加一个时间和实际完成的线，用来比较实际完成量和计划完成量中间的差距，跟踪甘特图从项目正式启动开始，每个时间段记录一次所以称为跟踪甘特图
